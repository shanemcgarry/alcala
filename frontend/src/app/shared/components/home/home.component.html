Introduction

Welcome to the Alcal치 Record Books. The Alcal치 Record Books are a digitisation of an 18th century book of accounts from the Royal Irish College in Alcal치 de Henares, Spain. Established in 1649, the college served as training for Catholic Priests from Ireland (Catholicism being outlawed in Ireland at the time). It existed until 1785 when it was absorbed into the Royal Irish College in Salamanca, Spain.

The records here specifically deal with the monetary accounts as recorded by the rector of the college from 1774 to 1781 (excluding 1780 which is missing from the original manuscript). The original manuscript is currently on display in the Russell Library at Maynooth University in Maynooth, Co. Kildare, Ireland.

The Digital Research Environment

This site is part of PhD thesis presented by Shane A. McGarry and the Maynooth University Department of Computer Science and has been funded in part through a grant from the Irish Research Council. The goal of this project is to create a Virtual Research Environment which allows the user construct queries through multiple modalities of engagement with the source material. This is primarily accomplished in 3 ways.

search Keyword Search

The keyword search provides the user with a standard search interface. Much like a Google search, the user can enter keywords and see a list of results. The keyword search also has an advanced search function that allows the user to specify additional criteria such as a specific year or to only include entries which contain the top n words in the corpus.

Go to Keyword Search

insert_chart_outlined Visualisation Search

The visualisation search provides the user with an alternative method of search engagement through the use of data visualisations. Here, the user can specify search criteria that is available in the standard keyword search, but rather than returning a list of results, the search is presented in the form of various types of charts. These charts can be manipulated, filtered, and clicked through to see the details of the data, allowing the user to eventually drill down to the individual manuscript page.

Go to Visualisation Search

dashboard Customisable Dashboard

The final component is that of the customisable dashboard which allows the user to not only create a custom dashboard to keep track of her favourite visualisations, but also build out a "story" that can help explain her particular research question. This component allows for a type of constructed knowledge that is an important factor in exploring these types of digitised manuscripts.
