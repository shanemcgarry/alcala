Data Curation

For each record, you can review the amounts and the primary words that make up each transaction. There is also a link to the actual page in the archive that will allow you to read the full transaction within the context of the other transactions on the page. In the "Categories" column, please provide a comma-delimited list of categories for the transaction. For example, if you wish to apply two categories to an item (e.g. "travel" and "food"), simply enter travel,food into the text box under "Categories" for that row. When you are finished, please click the "Save" button. NOTE: please try to keep the total number of categories limited as too many categories becomes difficult to visualise.
